# E-Commerce Sales & Customer Intelligence Dashboard

## 📌 Project Overview
An end-to-end data analytics and engineering project that processes raw e-commerce data to uncover business insights, track KPIs, and predict customer behavior. 

## 🏗️ Architecture & Tech Stack
* **Language:** Python (Pandas, Matplotlib, Seaborn)
* **Database:** MySQL
* **Visualization:** Power BI
* **Bonus (ML):** Scikit-Learn (K-Means Clustering)

## 🚀 How to Run the Project
1. Clone the repository: `git clone (https://github.com/akulgarg7/ecommerce-analytics-project)`
2. Create a virtual environment and activate it.
3. Install dependencies: `pip install -r requirements.txt`
4. Setup the database using the schema in `/sql/schema.sql`
5. Run the ETL pipeline: `python src/extract.py`
